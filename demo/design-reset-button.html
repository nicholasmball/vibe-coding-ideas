UX Spec v3: Text Reset Button Redesign

Counter App — Demo project · Prepared by UX Designer

Deliverables

  • Interactive demos showing all button states (default, hover, disabled, focus)
  • Amber colour scheme specification
  • Sizing & spacing tokens
  • Interaction model (hover glow, keyboard, disabled logic)
  • Component hierarchy with props
v3 redesign: User feedback rejected the icon-based circular button (v2). This version uses a text label ("Reset") with a warm amber colour accent.

1. Placement & Layout

Text button centered below the counter row. The rounded-rectangle shape matches the +/- buttons while the amber colour differentiates it as a utility action.

2. States

Default (count != 0)

Enabled

Amber ghost button. Warm colour draws attention without being aggressive.

Disabled (count === 0)

Disabled

25% opacity. HTML disabled attribute. No pointer events.

Hover (try it!)

Hover — amber glow

Background brightens, border strengthens, subtle amber glow shadow. Text shifts to brighter amber-400.

Focus (keyboard)

Focus-visible

Amber focus ring (#f59e0b) with 2px offset. Keyboard only via :focus-visible.

4. Keyboard & Accessibility

  • Tab order: [-] → [+] → [Reset]
  • No aria-label needed — "Reset" text is self-descriptive
  • Focus style: :focus-visible amber ring
  • Disabled: HTML disabled attribute removes from tab order
